功能介绍

功能概览

爱读RSS致力于提供完整而简洁的RSS阅读体验，每个功能都经过精心设计，确保您能够高效地获取和管理信息。

📡 RSS源管理

添加RSS源

  • 支持标准RSS 2.0和Atom 1.0格式
  • 自动检测和验证RSS源有效性
  • 智能解析RSS源标题、描述和图标
  • 支持HTTPS和HTTP协议

RSS源信息显示

  • 标题：显示RSS源的名称
  • 描述：显示RSS源的简介
  • 文章数量：显示未读/总文章数
  • 更新时间：显示最后更新时间
  • 状态指示：显示RSS源是否正常工作
  • 自定义图标：支持手动设置RSS源图标

📁 文件夹管理

组织功能

  • 创建多级文件夹结构
  • 拖拽方式移动RSS源
  • 文件夹重命名和删除
  • 文件夹展开/折叠状态记忆

智能分类

建议的文件夹分类方式：

  • 按主题：科技、新闻、娱乐、体育等
  • 按频率：每日必读、偶尔浏览等
  • 按来源：官方媒体、个人博客、社区等

📰 文章阅读

列表视图

  • 清晰的文章标题和摘要显示
  • 发布时间和作者信息
  • 未读/已读状态标识
  • 收藏状态显示
  • 文章来源RSS源标识

详细阅读页面

  • 优化的文章排版和字体渲染
  • 图片自适应显示
  • 链接点击跳转
  • 阅读进度记忆

快速操作

支持多种快速操作方式：

  • 滑动操作：左滑标记已读，右滑收藏
  • 长按菜单：显示完整操作选项
  • 批量操作：选择多篇文章进行批量处理

🔍 全文搜索功能

智能搜索

  • 在文章标题中搜索关键词
  • 在文章内容中搜索关键词
  • 在文章摘要中搜索关键词
  • 支持中英文搜索
  • 实时搜索结果更新
  • 搜索结果高亮显示

搜索索引

  • 自动索引：新文章自动建立搜索索引
  • 多字段索引：标题、正文、摘要独立索引
  • 性能优化：快速搜索响应，支持大量文章
  • 模糊匹配：支持部分关键词匹配

📤 数据导入导出

OPML格式支持

  • 完整支持OPML 1.0和2.0格式
  • 导出包含文件夹结构的完整订阅列表
  • 自动保存到下载目录，方便分享
  • 支持跨平台RSS阅读器数据迁移

导入功能

  • 批量导入：从OPML文件批量导入RSS源
  • 智能去重：自动检测重复RSS源，避免重复添加
  • 文件夹重建：自动创建OPML中的文件夹结构
  • 导入验证：验证RSS源有效性，确保导入成功

使用方法

导出订阅列表：

  1. 在RSS源列表页面右键点击空白区域
  2. 选择"导出OPML"
  3. 文件将保存到下载目录
  4. 文件路径会自动复制到剪贴板

导入订阅列表：

  1. 将OPML文件放入设备的下载目录
  2. 在RSS源列表页面右键点击空白区域
  3. 选择"导入OPML"
  4. 系统会自动选择最新的OPML文件
  5. 等待导入完成

🎨 图标管理系统

自动图标下载

  • RSS源添加时自动下载网站图标
  • 智能检测多种图标格式（ICO、PNG、JPG等）
  • 支持favicon.ico、apple-touch-icon等常见图标
  • 自动验证图标有效性，确保显示质量

手动图标设置

  • 自定义图标：支持手动设置任意图片URL作为图标
  • 图标预览：设置前可预览图标效果
  • 本地缓存：图标本地存储，离线时正常显示
  • 图标管理：支持重新下载或更换图标

使用方法

设置自定义图标：

  1. 右键点击RSS源
  2. 选择"设置图标"
  3. 输入图标URL（支持直接图片链接或网站地址）
  4. 点击确认，系统会自动下载并设置图标

⚙️ 自动刷新

智能更新

  • 可设置自动刷新间隔（15分钟到24小时）
  • 后台智能更新，不影响使用
  • 网络状态检测，WiFi优先更新
  • 更新失败自动重试机制

手动刷新

  • 下拉刷新：在文章列表页面下拉即可刷新
  • 单独刷新：可以单独刷新某个RSS源
  • 全部刷新：一键刷新所有RSS源

💾 数据管理

本地存储

  • 文章内容本地缓存，支持离线阅读
  • RSS源信息和设置本地保存
  • 阅读历史和收藏记录保存
  • 自动清理过期文章，节省存储空间
  • 搜索索引本地存储，快速搜索

数据安全

数据保护

所有数据都安全存储在本地设备中，建议定期导出OPML文件备份订阅列表，并记录重要的RSS源地址以防意外丢失。

🎨 个性化设置

外观设置

  • 浅色/深色主题切换
  • 字体大小调节（小、中、大、特大）
  • 行间距调节
  • 阅读背景颜色选择

阅读偏好

  • 列表模式：紧凑列表或卡片视图
  • 排序方式：按时间、标题或RSS源排序
  • 显示选项：是否显示摘要、图片等

性能优化

爱读RSS采用了多项性能优化技术，包括：

  • 异步加载，确保界面流畅
  • 图片懒加载，节省网络流量
  • 智能缓存策略，提升响应速度
  • 内存优化，减少资源占用
  • 搜索索引优化，快速全文搜索

🛡️ 隐私保护

数据安全

  • 所有数据本地存储，不上传到服务器
  • 不收集用户个人信息
  • 不追踪用户阅读行为
  • 支持完全离线使用
