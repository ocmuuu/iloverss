快速开始

欢迎开始您的RSS阅读之旅！

本指南将帮助您快速上手爱读RSS，让您在几分钟内就能开始享受高效的信息阅读体验。

第一步：添加RSS源

RSS源是您获取信息的来源，就像订阅报纸一样。

方法一：手动添加

  1. 打开爱读RSS应用
  2. 点击主界面右上角的 "+" 按钮
  3. 在弹出的对话框中输入RSS源地址
  4. 点击 "添加" 按钮
  5. 等待应用验证并获取RSS信息
示例RSS地址： https://www.ruanyifeng.com/blog/atom.xml https://sspai.com/feed https://www.zhihu.com/rss

方法二：从浏览器添加

  1. 在浏览器中访问您喜欢的网站
  2. 寻找RSS图标或"订阅"链接
  3. 复制RSS地址
  4. 回到爱读RSS，按照方法一添加

小贴士：如何找到RSS地址

  • 很多网站在页面底部或侧边栏有RSS链接
  • 博客类网站通常支持RSS订阅
  • 新闻网站的RSS地址通常是：网站域名/rss 或 /feed

第二步：管理RSS源

创建文件夹

为了更好地组织您的RSS源，建议创建文件夹进行分类：

  1. 点击主界面的 "文件夹管理" 按钮
  2. 点击 "新建文件夹"
  3. 输入文件夹名称（如：科技、新闻、博客等）
  4. 点击 "确定" 保存

移动RSS源到文件夹

  1. 长按想要移动的RSS源
  2. 在弹出菜单中选择 "移动到文件夹"
  3. 选择目标文件夹
  4. 点击 "确定"

第三步：阅读文章

浏览文章列表

  • 点击RSS源查看该源的文章列表
  • 未读文章会显示为粗体
  • 点击文章标题进入详细阅读页面

文章操作

在文章列表中，您可以通过滑动手势进行快速操作：

  • 向左滑动：标记为已读/未读
  • 向右滑动：添加到收藏/取消收藏
  • 长按文章：显示更多操作选项

第四步：个性化设置

调整刷新频率

  1. 进入 "设置" 页面
  2. 找到 "自动刷新" 选项
  3. 选择合适的刷新间隔（建议：每小时或每2小时）

阅读偏好设置

  • 字体大小：调整阅读时的字体大小
  • 主题模式：选择浅色或深色主题
  • 阅读模式：选择列表视图或卡片视图

恭喜！您已经掌握了基本操作

现在您可以开始享受RSS阅读的乐趣了。记住，好的RSS源是高质量阅读体验的关键，建议您慢慢积累自己喜欢的信息源。

注意事项

  • 某些RSS源可能需要网络连接才能正常更新
  • 建议不要添加过多RSS源，以免信息过载
  • 定期清理不再关注的RSS源，保持列表整洁
