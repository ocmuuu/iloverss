快捷键一览

文章导航

  • J / 右方向键 → 下一篇文章
  • K / 左方向键 → 上一篇文章

阅读操作

  • M → 切换已读 / 未读
  • S → 收藏 / 取消收藏

订阅源导航

  • Shift + N → 选择下一个订阅源
  • Shift + P → 选择上一个订阅源

批量与刷新

  • Shift + A → 当前源全部标记已读
  • R → 刷新全部订阅源

界面操作

  • H → 折叠 / 展开订阅源侧栏

提示

快捷键在文章列表与阅读界面均可使用，为确保触发，请先点击窗口任意位置以获得键盘焦点。
