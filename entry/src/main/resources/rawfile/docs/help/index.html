欢迎使用爱读RSS

感谢您选择爱读RSS！这是一款专为鸿蒙系统设计的RSS阅读器，致力于为您提供简洁、高效的阅读体验。

主要特性

  • 支持标准RSS/Atom格式
  • 智能文章解析与排版
  • 离线阅读支持
  • 文件夹分类管理
  • 全文搜索功能
  • OPML导入导出
  • 图标管理功能
  • 深色模式支持
  • 本地数据存储

    • 系统要求

    运行环境

    • 操作系统：HarmonyOS 4.0 或更高版本
    • 存储空间：建议预留100MB可用空间
    • 网络连接：需要网络连接以同步RSS内容
    • 权限要求：网络访问、存储读写权限

    使用技巧

    提升使用效率的小技巧

    • 快速添加RSS源：点击右上角的"+"按钮，输入RSS地址即可
    • 文件夹管理：右键点击空白区域，选择"添加文件夹"创建分类
    • 搜索文章：使用搜索功能快速找到感兴趣的内容
    • 备份订阅：定期导出OPML文件备份您的订阅列表
    • 自定义图标：为RSS源设置个性化图标，提升视觉体验
    • 标记收藏：重要文章可以收藏保存，方便后续查看
