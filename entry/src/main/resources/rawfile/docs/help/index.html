欢迎使用爱看RSS

感谢您选择爱看RSS！这是一款专为鸿蒙系统设计的RSS阅读器，致力于为您提供简洁、高效的阅读体验。

主要特性

  • 支持标准RSS/Atom格式
  • 智能文章解析与排版
  • 离线阅读支持
  • 文件夹分类管理
  • 全文搜索功能
  • 深色模式支持
  • 本地数据存储

    • 系统要求

    兼容性说明

    • HarmonyOS 5.0 及以上版本
    • 建议可用存储空间 100MB 以上
    • 网络连接（用于获取RSS内容）

    快速上手

    1. 添加RSS源：点击主界面的"+"按钮，输入RSS地址
    2. 创建文件夹：长按RSS源可移动到文件夹进行分类
    3. 阅读文章：点击文章标题进入详细阅读页面
    4. 标记状态：滑动文章可标记为已读/未读或收藏

    小贴士

    您可以在设置中调整刷新频率、阅读字体大小等个性化选项，获得更好的使用体验。
